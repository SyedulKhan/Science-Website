		var duration = 910; 			// tenths of a second;
		var warningThreshold = 600; 	// 60 seconds
		var dangerThreshold = 200; 		// 20 seconds
		
        var durationRemaining = duration;

        var score = 0;
        var correct = 0;
        var incorrect = 0;

        startQuiz();
        
        var timer;       

        function timerTick() {

            var tDisplay = document.getElementById('TimeDisplay');

            if(durationRemaining >= 1){
                durationRemaining--;
            }
            else{
                endQuiz();
            }

            if (durationRemaining <= dangerThreshold) {
                tDisplay.className = "Timer TimerDanger";
            } else if (warningThreshold <= 600) {
                tDisplay.className = "Timer TimerWarning";
            }
            else {
                tDisplay.className = "Timer";
            }
           
            tDisplay.innerText = (durationRemaining/10).toFixed(1) + " seconds";
        }

        function startQuiz(){            
            timer = setInterval(timerTick, 100);
        }

        function endQuiz(){
            clearInterval(timer);
            // score = getScore();
        }

    </article>
    <article class="SciPageAuthor" style="margin-top:20px;">

    <article class="SciSection">
        <article class="SciSectionTitle">
            History
        </article>
        <article class="SciSectionText">
            <p>Although Albert Einstein often gets recognition for predicting gravitational waves based on his renouned theories of relativity and fields, the possibility was in fact previously discussed by Oliver Heavside as far back as 1893<sup><a href="#grav_ref02b" id="grav_ref02a" class="Reference">[2]</a></sup>. 
            Another mention of note is Henri Poincaré, in 1905, who first proposed gravitational waves emanating from a body and propogating at the speed of light. Only later in 1916 did Einstein predict gravitational waves that transport energy as gravitational radiation. </p>
            <p>On 11<sup>th</sup> February 2016, The LIGO Scientific Collaboration (LSC) announced they had detected gravitational waves.</p>
        </article>
    </article>
    <article class="SciSection">
        <article class="SciSectionTitle">
            Theory
        </article>
        <article class="SciSectionText">
            <p>
                The principle behind Newtonain gravity is that each body of matter has a gravitational property proportional to its mass, an effect of this property is that bodies of matter are drawn to it. The closer the matter is the stronger the affect of its gravity.
            </p><p>Einstein&apos;s theory adds that the effects of gravity results from the matter&apos;s affect on the curvature of spacetime, the higher the mass, the greater the curvature and the greater the curvature the greater the gravity. Accelerating objects can generate changes in the curvature of space time, a good analogy would be the waves on water created by the wake of a boat. From properties of the waves such as the frequency and amplitude, it is possible to tell the amount of distortion caused by the object allowing us to calculate its size and mass.</p>
        </article>
    </article>
    <article class="SciSection">
        <article class="SciSectionTitle">
            Research
        </article>
        <article class="SciSectionText">
            <p>Research into the theory has been carried out by many notable scientists over the years, but the first gravitational waves to be detected were by The LIGO Scientific Collaboration (LSC) 14<sup>th</sup> September 2015 (to be later announced on 11<sup>th</sup> February 2016). They have continued to detect additional gravitational waves.</p>
        </article>
    </article>
    <article class="SciSection">
        <article class="SciSectionTitle">
            Implications
        </article>
        <article class="SciSectionText">
            <p>While previously having a view of the universe limited to objects emitting electromagnetic radiation that were not travelling away from earth at the speed of light, the detection of gravitational waves allows observers to recieve metrics from billions of years in the past, billions of lightyears away!</p>
            <p>This will undoubtedly further our knowledge of our universe and the theoretical spacial occurences that have previously been out of our reach. On a personal note, I suspect that this closer look at the theories of gravity may potentially result in a significant furthering of multi-dimensional theories, hopefully including what other effects the supposedly weakest of the fundamental forces maybe having outside our of perceivable dimensions.</p>
        </article>

<div class="SciPage">
    <div class="SciPageTitle" style="text-align:center;">Higgs boson: From theory to discovery</div>

    <div class="SciSection">
        <div class="SciSectionTitle" id="background">
            Background
        </div>
        <div class="SciSectionText">
            <p>
                Back in the 1970s, scientists and researchers realised that there are
                correlations between two of the four fundamental forces; the electromagnetic
                force and the weak force. Those two forces were analysed into theory and formed
                the grounds of the Standard Model. This Model belongs to particle physics, a field
                driven by physicists dedicated to breaking down the endlessly complicated
                universe to its most fundamental building blocks. The relationship between the
                electromagnetic and weak force implied that electricity, magnetism, light and some
                types of radioactivity are all manifestations of a single underlying force known
                as the electroweak force.
            </p>
            <p>
                The equations and theorems that described this force and its particles, the
                photon and the <a href="https://en.wikipedia.org/wiki/W_and_Z_bosons" target="_blank" class="Reference">W and Z bosons</a>, were all correct apart from a big problem; all
                these particles emerged without a mass. Now that would not be a problem for the
                photon, but the W and Z have mass. This is where theoretical physicists
                <a href="https://en.wikipedia.org/wiki/Robert_Brout" target="_blank" class="Reference">Robert
                Brout</a>, 
                <a href="https://en.wikipedia.org/wiki/Fran%C3%A7ois_Englert" target="_blank" class="Reference">Francois Englert</a> 
                and <a href="https://en.wikipedia.org/wiki/Peter_Higgs" target="_blank" class="Reference">Peter Higgs</a> came in to solve this problem. They
                suggested a mechanism, which took the name Brout-Englert-Higgs mechanism
                afterwards, that gives a mass to the W and Z when they interact with an invisible
                field, now called the Higgs field, which fills the universe. The Higgs Field,
                like most fundamental fields, should have an associated particle – the Higgs
                boson. That boson, when found, would be a visible manifestation of the Higgs
                field.
            </p>
        </div>
    </div>
    <div class="SciSection">
        <div class="SciSectionTitle" id="preparation">
            Preparation
        </div>
        <div class="SciSectionText">
            <p>
                To implement and put in practise all these years of questioning, reasoning and
                theorems, a colossal source of experimental energy should exist to make such goals
                even observable.  For this reason, but for many other as well, the famous Large Hadron
                Collider was built. For those that have not heard of it, Large Hadron Collider is a
                particle collider built by <acronym title="European Organization for Nuclear Research">CERN</acronym>. To construct this, it took 10 years and 10.000
                scientists and engineers. Hence, one could argue that <acronym title="Large Hadron Collider">LHC</acronym> is the most complex and
                sophisticated experimental facility ever built.
            </p>
            <p>
                The collider is equipped with superconducting magnets and accelerating structures
                that are required to boost the speed of the colliding particles. In order to avoid
                the loss of energy and achieve minimal electrical resistance these magnets need to
                be cooled down to -271.3&deg;C and for that reason the <acronym title="Large Hadron Collider">LHC</acronym> is being
                supplied with liquid helium. The accelerator contains two pipes; each is filled with
                high-energy particle beams that are being accelerated in opposite directions until
                they reach a speed that is close to that of the speed of light. Making that happen
                requires keeping those two pipes at ultrahigh vacuum. Then the two beams, being under
                control, are made to collide.
            </p>
            <p>
                Many experiments were conducted with no desired outcome throughout the starting years
                of <acronym title="Large Hadron Collider">LHC's</acronym> function, but the scientists that work there reported helpful evidence
                every time.
            </p>
        </div>
    </div>
    <div class="SciSection">
        <div class="SciSectionTitle" id="discovery">
            Discovery
        </div>
        <div class="SciSectionText">
            <p>
                On the 4<sup>th</sup> of July 2012, two experiments that were being conducted at 
                <acronym title="European Organization for Nuclear Research">CERN’s</acronym>
                <acronym title="Large Hadron Collider">LHC</acronym>
                announced they had each observed a new particle that possesses a mass of approximately
                126 giga-electron volts (1 GeV = 1 billion electron volts). This particle consisted
                with the Higgs boson that was predicted by the Standard Model.
            </p>
            <p>
                A year later, Francois Englert and Peter Higgs were awarded with the Nobel prize in
                physics for their theoretical work that contributed in understanding the origin of
                mass of subatomic particles. You can see the nomination video 
                <a href="https://www.youtube.com/watch?v=aBBOTC0xg4s" target="_blank" class="Reference">here</a>.
            </p>
        </div>
    </div>

        <div class="SciSectionTitle" id="considerations">
            Considerations
        </div>
        <div class="SciSectionText">
            <p>
                There are researchers that argue that even though the Higgs boson that was discovered
                back in 2012 may indeed look like what they were looking for, it is not exactly that.
                Looking back to our history we have seen cases that theories became realities and
                remain so but we have also witnessed cases that theories were made into beliefs that
                lasted centuries and afterwards were disproved. The latter case though, was not caused
                by misguidance but because of insufficient evidence paired together with the
                non-perishable impatience of the humankind. In which categories of the two does the
                Higgs discovery belong to? Only a few might be able to really tell but even so it is
                too early to say. Not to mention that there are many other theorems related to Higgs
                boson that are waiting to be tested upon in <acronym title="Large Hadron Collider">LHC</acronym>.
            </p>
            <p>
                Finding such a particle or any particle for that
                matter is like a person that clapped too early in a special event; at that moment,
                nobody knows if it was the right action but that single clap could result to the
                whole audience clapping. Higgs boson or not, the people that are pursuing and devoting
                their lives to this, certainly deserve our gratitude for trying to solve one of the

    <p> Definition - Where one object can be a part of more than one category of object.</p>
    <div class="Body">
        <h3>What is polymorphism?</h3>
        <div>
            <img src="" align="Left" alt="polymorphism image" width="200" height="200" style="float:right; clear:none;">
        </div>
        <p> This would be a basic image of what polymorphism is and how it works. This means that a variable that stores a Vehicle object, can store any descendant of the Vehicle class.</p>
        <p>Example:</p>
        <ul>

function changeFontSize(change) {
    var elemSize = document.getElementsByClassName("FontSize");
    var fontSizes = ["xx-small", "x-small", "small", "medium", "large", "x-large"];
    var fontSizeAbr = ["XXS", "XS", "S", "M", "L", "XL"];
    var hasValue = false;
    var sizeChar;
    var size = elemSize[0].style.fontSize;

    if (size == "x-large" && change==1) {
        alert("This is the maximum font size");
    }
    else if (size == "xx-small" && change==-1) {
        alert("This is the minimum font size");
    }
    else {
        for (var j = 0; j < fontSizes.length; j++) {
            if (size == fontSizes[j]) {
                hasValue = true;
                var newSize = fontSizes[j + change];
                sizeChar = fontSizeAbr[j + change];
                break;
            }
        }
        if (!hasValue) {
            if (change == 1) {
                newSize = "large";
                sizeChar = "L";
            }
            else {
                newSize = "small";
                sizeChar = "S";
            }
        }
        document.getElementById("currentFontSize").innerHTML = sizeChar;
        for (var i = 0; i < elemSize.length; i++) {
            elemSize[i].style.fontSize = newSize;
        }
    }
}

// Loads an HTML document into an element as its inner HTML.
function loadContent(path, elementId, showMenu)
{
    try
    {
        // Prepare for HTTP Request.
        var c = new XMLHttpRequest();

        // Requests HTML file.
        c.open("GET", path, false);
        c.send();

        // Returns the response as HTML and sets it to the inner HTML of the specified element
        document.getElementById(elementId).innerHTML = c.response;

        // Makes the menu visible/invisible.
        if(showMenu) document.getElementById("page_links").setAttribute("class", "Nav VNav VNavMain Dark-dark Visible");
        else document.getElementById("page_links").setAttribute("class", "Nav VNav VNavMain Dark-dark Invisible");
    }
    catch (e)
    {
        // document.getElementById("page_content").innerHTML = "<h1>Error: " + e.message + "</h1>";
        throw e;
    }
}

    var menuUrl = "./menus/home_menu.html";
           
    pageUrl = "./pages/" + pageName + ".html";
    menuUrl = "./menus/" + pageName + "_menu.html";

        loadContent(pageUrl, 'page_content', false);
    } catch (e) {
        // Return to home page on exception if possible.
        if (pageName != 'home')
            showPage('home', true);
    }
    try{
        if (showMenu) loadContent(menuUrl, 'page_links', true);
    } catch (e) {
       // Do nothing.
    }
}

        try {
            document.getElementById("orderedstickers").innerHTML = totalStickers + " Plutonium stickers : £" + individualProductTotal[2];
        }

    }

//Calculates the order total (price)
function calcTotal(prodNo, quantity) {
    switch (prodNo) {
        case 1: orderTotal += quantity * 15;
            break;
        case 2: orderTotal += quantity * 10;
            break;
        case 3: orderTotal += quantity * 25;
            break;
        case 4: orderTotal += quantity * 5;
            break;
        case 5: orderTotal += quantity * 4;
            break;
        case 6: orderTotal += quantity * 30;
            break;
        default:
            break;
    }
    charityCalc(orderTotal);
    document.getElementById("tamount").innerHTML = "£ " + orderTotal;
}

//Validation for the checkout form (personal details)
function validate(frm) {

    var emailFormat = /^([A-Za-z0-9_\-\.])+\@([A-Za-z0-9_\-\.])+\.([A-Za-z]{2,4})$/;
    var radiobuttons = document.getElementsByClassName("CardType");
    var uncheckedRadioButtons = 0;
    var defGood = false;
    var allGood = new Array(15);

    if (frm.title.value == "-") {
        allGood[0] = styleInputForm(false, "customertitle");
    } else {
        allGood[0] = styleInputForm(true, "customertitle");
    }

    if (frm.firstname.value.length < 3 || !frm.firstname.value.match(/^[a-zA-Z ]+$/)) {
        allGood[1] = styleInputForm(false, "customerfirstname");
    } else {
        allGood[1] = styleInputForm(true, "customerfirstname");
    }

    if (frm.lastname.value.length < 4 || !frm.lastname.value.match(/^[a-zA-Z]+$/)) {
        allGood[2] = styleInputForm(false, "customerlastname");
    } else {
        allGood[2] = styleInputForm(true, "customerlastname");
    }

    if (emailFormat.test(frm.email.value) == false) {
        allGood[3] = styleInputForm(false, "customeremail");
    } else {
        allGood[3] = styleInputForm(true, "customeremail");
    }

    if (!frm.number.value == "") {
        if (frm.number.value.length < 10 || !frm.number.value.match(/^[0-9]+$/)) {
            allGood[4] = styleInputForm(false, "customernumber");
        } else {
            allGood[4] = styleInputForm(true, "customernumber");
        }
    } else {
        allGood[4] = styleInputForm(true, "customernumber");
    }

    if (frm.address1.value.length < 8) {
        allGood[5] = styleInputForm(false, "customeraddress1");
    } else {
        allGood[5] = styleInputForm(true, "customeraddress1");
    }

    if (!frm.address2.value == "") {
        if (frm.address2.value.length < 4) {
            allGood[6] = styleInputForm(false, "customeraddress2");
        } else {
            allGood[6] = styleInputForm(true, "customeraddress2");
        }
    } else {
        allGood[6] = styleInputForm(true, "customeraddress2");
    }

    if (frm.postcode.value.length < 5 || !frm.postcode.value.match(/\d+/g) || !frm.postcode.value.match(/^[a-zA-Z]/) || !frm.postcode.value.match(/[a-zA-Z]$/) || !frm.postcode.value.match(/[a-zA-Z0-9 *]/)) {
        allGood[7] = styleInputForm(false, "customerpostcode");
    } else {
        allGood[7] = styleInputForm(true, "customerpostcode");
    }

    if (frm.city.value.length < 4 || !frm.city.value.match(/^[a-zA-Z]+$/)) {
        allGood[8] = styleInputForm(false, "customercity");
    } else {
        allGood[8] = styleInputForm(true, "customercity");
    }

    for (var i = 0; i < radiobuttons.length - 3; i++) {
        if (radiobuttons[i].checked) {
            allGood[9] = styleInputForm(true, "visadebit");
            allGood[9] = styleInputForm(true, "visacredit");
            allGood[9] = styleInputForm(true, "mastercard");
            break;
        }
        else {
            uncheckedRadioButtons++;
        }
    }

    if (uncheckedRadioButtons == 3) {
        allGood[9] = styleInputForm(false, "visadebit");
        allGood[9] = styleInputForm(false, "visacredit");
        allGood[9] = styleInputForm(false, "mastercard");
    }

    if (frm.cardholder.value.length < 6 || !frm.cardholder.value.match(/^[a-zA-Z ]+$/)) {
        allGood[10] = styleInputForm(false, "cardholder");
    } else {
        allGood[10] = styleInputForm(true, "cardholder");
    }

    if (frm.cardnumber.value.length < 16 || !frm.cardnumber.value.match(/^[0-9]+$/)) {
        allGood[11] = styleInputForm(false, "cardnumber");
    } else {
        allGood[11] = styleInputForm(true, "cardnumber");
    }

    if (frm.expirymonth.value == "-") {
        allGood[12] = styleInputForm(false, "expirymonth");
    } else {
        allGood[12] = styleInputForm(true, "expirymonth");
    }

    if (frm.expiryyear.value == "-") {
        allGood[13] = styleInputForm(false, "expiryyear");
    } else {
        allGood[13] = styleInputForm(true, "expiryyear");
    }

    if (frm.expirymonth.value < 4 && frm.expiryyear.value == 2017) {
        allGood[12] = styleInputForm(false, "expirymonth");
        allGood[13] = styleInputForm(false, "expiryyear");
        alert("Your card has expired");
    }

    if (!frm.termsandconditions.checked) {
        allGood[14] = styleInputForm(false, "termsandconditions");
    } else {
        allGood[14] = styleInputForm(true, "termsandconditions");
    }

    var falseCounter = 0;
    for (var i = 0; i < allGood.length; i++) {
        if (allGood[i] == false) {
            falseCounter++;
        }
    }
    if (falseCounter == 0) {
        messageOrder(frm);
    }
}
